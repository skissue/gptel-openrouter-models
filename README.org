#+title: gptel-openrouter-models
#+author: Ad <me@skissue.xyz>

A very simple package that provides an autogenerated list of all [[https://openrouter.ai][OpenRouter]] models for [[https://github.com/karthink/gptel][gptel]], with full metadata (capabilities, pricing, context window). The model list is updated daily from OpenRouter's API via GitHub Actions.

* Usage
Clone/install the package, then use configuration like the following to create an OpenRouter backend:

#+begin_src emacs-lisp
(gptel-make-openai "OpenRouter"
  :host "openrouter.ai"
  :endpoint "/api/v1/chat/completions"
  :key 'your-openrouter-key
  :models (gptel-openrouter-models))
#+end_src
